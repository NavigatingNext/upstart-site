# Upstart Static Site

- Open `index.html` in a browser to preview.
- Replace logos/photos in `/assets` and update text as needed.
- All links are local; deploy anywhere you can host static files.

Pages:
- index.html (Home)
- students.html
- parents.html
- sponsors.html
- bios.html (Student Stories)
- news.html (News & Trends with reputable links)
- resources.html (External programs & partners)
- about.html
- contact.html (Application form stub)
